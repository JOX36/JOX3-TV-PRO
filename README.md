# JOX3 TV 🎬

App nativa de Android para reproducción IPTV con diseño futurista en cyan y azul eléctrico.

## Características

- 🏠 **Home** con carrusel de recomendación y último visto
- 📺 **TV en Vivo** con vista de grilla y lista
- 🎬 **Películas** con categorías y búsqueda
- 📺 **Series** con temporadas y episodios
- ❤️ **Favoritos** por tipo de contenido
- 🔍 **Búsqueda** global
- ⚙️ **Ajustes** con multi-lista de servidores
- 🎮 **Compatible con TV Box** y control remoto
- 📱 **PiP** (Picture in Picture)
- 🔊 **Selección de audio y subtítulos**
- 🎨 **Diseño futurista** cyan/azul eléctrico

## Requisitos

- Android 5.0+ (API 21)
- Lista IPTV con credenciales Xtream Codes

## Instalación

1. Ve a la pestaña **Actions** en GitHub
2. Selecciona el último workflow **Build APK**
3. Descarga el artefacto `StreamVault-debug`
4. Instala el APK en tu dispositivo

## Uso en TV Box

La app detecta automáticamente si se ejecuta en un TV Box y muestra la interfaz optimizada para control remoto:

- **D-pad Navegar** entre elementos
- **OK/Enter** Seleccionar
- **←/→** Buscar adelante/atrás en el reproductor
- **↓** Abrir grilla de canales
- **Reproducir/Pausar** Controles de media

## Configuración

1. Abre la app
2. Ve a **Ajustes** → **Agregar nueva lista**
3. Ingresa los datos de tu servidor Xtream Codes
4. ¡Listo! El contenido se cargará automáticamente
